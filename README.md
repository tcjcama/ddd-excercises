#Overview
This is a simple project to practice DDD and best practices.

##Setup

1. clone repo

```
clone repo https://github.com/tcjcama/ddd-excercises.git
```

2. copy the base phpunit config

```
cp phpunit.dist.xml phpunit.xml 
```

3. **Enjoy** !! 🍫

## Depelopment 🛠

Run tests

```
./vendor/bin/phpunit
```
